Станьте премиум клиентом

Нашим премиум клиентам предоставляется набор для комфортного перелета, включающий тапки, крем, маску для лица и подарочную карту. Тапки обеспечивают удобство и защиту ног во время полета, крем помогает увлажнить кожу, маска для лица позволяет расслабиться во время полёта, а подарочная карта даст возможность насладиться дополнительными услугами компании во время путешествия.
Премиум статус получается при осуществлении десяти заказов на нашем сайте.
